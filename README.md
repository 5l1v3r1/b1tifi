### Description
The Botdr4g0n is a security tool for DDOS attack on SSH BOT management for distributed attacks.


### Installation
- python 2.7

```bash
git clone https://github.com/mh4x0f/botdr4g0n.git
cd botdr4g0n
pip install -r requirements.txt
```

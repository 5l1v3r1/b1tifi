### Description
The Botdr4g0n is a focused tool for DDOS attack on SSH BOT management and installation tools for distributed attacks.


### Installation
```bash
git clone https://github.com/mh4x0f/botdr4g0n.git
cd botdr4g0n
pip install -r requirements.txt
```
